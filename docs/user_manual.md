# User Manual

## Installation

1. Open Google Chrome.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** via the toggle switch in the top right.
4. Click the **Load unpacked** button in the top left.
5. Select the `chrome_plugin_saveaspng` directory.

## Usage

1. Navigate to the webpage you wish to capture.
2. Click the extension icon in the Chrome toolbar.
   - *Tip: You may want to "pin" the extension to keep the icon visible.*
3. A popup will appear in Dark Mode.
4. Click either **Save as PNG** or **Save as JPG**.
5. Do not interact with the page while the capture is taking place. The extension will automatically scroll through the page.
6. Once complete, the captured image will be downloaded to your default Downloads folder.
