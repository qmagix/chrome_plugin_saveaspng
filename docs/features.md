# App Features

## Page Capture

- **Full Page Screenshot**: Captures the entirety of the webpage by scrolling and stitching screenshots silently.
- **Multiple Formats**: Allows saving the resulting capture as either a PNG or a JPG file.

## User Interface

- **Dark Mode**: By default, the plugin popup employs a dark theme with modern aesthetics, providing a seamless visual experience.
- **Intuitive Controls**: Clear buttons for "Save as PNG" and "Save as JPG" make usage frictionless.
